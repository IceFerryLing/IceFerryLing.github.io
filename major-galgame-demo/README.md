# 哈工深 25届专业选择 Galgame Demo

这是一个围绕“哈工深 25届工科试验班：计算机与信息类专业选择”制作的静态网页 Galgame Demo。玩家可以在五位代表不同专业方向的角色之间进行选择，通过对话和实验室事件了解专业特点，并获得对应的专业倾向结局。

## 运行方式

直接用浏览器打开 `index.html` 即可运行，不需要安装依赖或启动服务器。

## 项目结构

```text
major-galgame-demo/
├─ index.html          页面结构与角色入口
├─ styles.css          场景、角色、对话框和响应式样式
├─ script.js           通用渲染、小游戏引擎、选择逻辑、存档逻辑
├─ characters/         五位角色的独立资料文件夹
│  ├─ cs/profile.js    洛泠资料、剧情、小游戏配置
│  ├─ ee/profile.js    栖禾资料、剧情、小游戏配置
│  ├─ auto/profile.js  青岚资料、剧情、小游戏配置
│  ├─ ic/profile.js    微澜资料、剧情、小游戏配置
│  └─ comm/profile.js  星遥资料、剧情、小游戏配置
├─ png/                当前使用的 PNG 人物图像
└─ assets/characters/  旧版 SVG 角色资源
```

## 已实现功能

- 五个专业分支：计算机、电气、自动化、集成电路、通信工程
- 使用 PNG 人物图像作为角色立绘
- 选中角色后，该角色会放大并占据主要视觉位置
- 每位角色拥有身份背景、性格特色、关键词和个人目标档案
- 五位角色资料、对话分支、小游戏配置分别放在 `characters/*/profile.js`，便于单独修改
- `script.js` 会自动汇总 `window.characterData` 中的角色模块，主脚本只保留通用玩法逻辑
- 每个专业拥有“初选介绍 -> 理论对谈/角色小游戏 -> 实验室事件 -> 方向确认 -> 多方向结局”的剧情流程
- 每位角色增加了专业理论讨论选项，可进一步了解相关知识框架
- 每个专业至少拥有两个结局倾向，玩家可在实验室事件后选择不同发展方向
- 每条角色线加入更青春恋爱向的对话氛围，直线流程不少于 5 轮对话
- 五个角色小游戏：
  - 洛泠：输入 `HIT` 的 ASCII 二进制编码，启动导览 AI
  - 栖禾：平衡光伏、电池与电网供给
  - 青岚：操控巡检机器人避障到达目标
  - 微澜：在版图网格中铺设金属线，绕过 IP 宏单元接通 PAD 与 CORE
  - 星遥：选择稳定的无线链路配置
- 打字机式对话展示
- 理性、热情、协作三项倾向数值
- 本地 `SAVE` / `LOAD` 存档
- `CONFIG` 设置：文字速度、背景动效开关
- 移动端基础响应式布局

## 玩法说明

1. 进入页面后，从五位角色中选择一个感兴趣的专业方向。
2. 被选中的角色会成为画面主视觉，并展开该专业的介绍剧情。
3. 阅读人物档案，可选择理论对谈或直接进入该角色对应的专业小游戏。
4. 完成小游戏后继续实验室事件，了解该专业的学习和应用场景。
5. 在实验室事件后进入方向确认对话，再选择不同发展方向进入对应专业倾向结局。
6. 可以通过 `SAVE` 保存当前进度，通过 `LOAD` 读取存档。

## 可扩展方向

- 为每个专业增加更多中途选择和多结局
- 根据理性、热情、协作数值生成综合推荐结局
- 增加“专业图鉴”或“知识卡片”页面
- 增加新生视角的校园地图探索玩法
